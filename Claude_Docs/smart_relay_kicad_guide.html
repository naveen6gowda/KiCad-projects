SHEET 1 KiCad Project Setup & Libraries Do this once before drawing
1
Create a new KiCad project
Open KiCad → File → New Project. Name it smart_relay_xiao. Accept the default folder structure.
2
Add Seeed XIAO library (if not already present)
Open the Schematic Editor → Preferences → Manage Symbol Libraries.
Click + (Add). Set Name = Seeed_XIAO.
https://github.com/Seeed-Studio/OPL_Kicad_Library
Clone/download that repo. Point the library path to Seeed_XIAO_ESP32C6.kicad_sym.
⚠ "Library already exists" warning is harmless — click OK and continue. The library loads correctly.
3
Verify built-in libraries are active
Ensure these are enabled in Manage Symbol Libraries → Global Libraries:
  • Device — resistors, caps, diodes, transistors
  • Relay_THT — SRD-05VDC-SL-C relay
  • Isolator — PC817
  • Regulator_Linear — AMS1117-3.3
  • Connector — terminal blocks
4
Set up hierarchical sheets
In the root schematic, use Place → Hierarchical Sheet to create 7 sub-sheets:
psu.kicad_sch · mcu.kicad_sch · relay_ch1.kicad_sch · relay_ch2.kicad_sch · energy_monitor.kicad_sch · mains_input.kicad_sch

Alternatively: you can draw everything on a single sheet — use Net Labels to connect distant parts.
5
Add a title block
File → Page Settings. Set:
Title: Smart Relay PCB · Rev: 1.0 · Date: today · Company: your name
SHEET 2 Power Supply — HLK-PM01 + AMS1117-3.3 psu.kicad_sch
ℹ HLK-PM01 produces an isolated 5VDC output from 230VAC mains. AMS1117-3.3 steps 5V down to 3.3V for the ESP32.
  MAINS_L ─────────────────┐
                        [HLK-PM01]  230VAC→5VDC
  MAINS_N ─────────────────┘    │
                                 │ +5V ──┬────── +5V net label
                                 │       │
                                 ⏚      [C1 100µF/10V] to GND
                                         │
                                +5V ─────┤IN  [AMS1117-3.3]  OUT ─── +3V3 net label
                                         │TAB/ADJ to GND
                                         │
                                       [C2 10µF] [C3 100nF]  both to GND
1
Place HLK-PM01 symbol
Press A → search HLK-PM01.
If not found, use a generic 4-pin power module or a custom symbol with pins: L, N, +Vo, -Vo (GND).
Place it on the left side of the sheet.
2
Connect HLK-PM01 mains pins to net labels
Add net labels: press L.
• L pin → net label MAINS_L
• N pin → net label MAINS_N
• -Vo (GND) pin → Power symbol GND (press P, search GND)
3
Add 5V output decoupling capacitor C1
Press A → Device:C. Place near HLK-PM01 +Vo pin.
Value: 100µF 10V · Ref: C1
Connect: + pin to +5V net · − pin to GND
4
Place AMS1117-3.3 LDO (SOT-223)
Press A → search AMS1117-3.3 in Regulator_Linear.
Place to the right of HLK-PM01.
• IN pin → wire → +5V net label
• GND / ADJ pin → GND power symbol
• OUT pin → net label +3V3
5
Add AMS1117 input & output capacitors
C2 (10µF ceramic) — on AMS1117 IN pin to GND
C3 (100nF ceramic) — on AMS1117 OUT (+3V3) to GND
C4 (10µF ceramic) — on AMS1117 OUT (+3V3) to GND
⚠ AMS1117 thermal tab (on SOT-223) must connect to GND. Assign footprint Package_TO_SOT_SMD:SOT-223-3_TabPin2 which ties the tab to pin 2 (GND).
6
Add power flags to suppress ERC warnings
Press P → search PWR_FLAG.
Place one on the +5V net and one on +3V3 net (both connected to the respective net). This tells ERC these nets are driven.

SHEET 3 MCU — XIAO ESP32-C6 (DIP) mcu.kicad_sch
1
Place XIAO ESP32-C6 symbol
Press A → search XIAO-ESP32-C6-DIP in Seeed_XIAO library.
Place centrally on the sheet. This module has 14 pins (7 per side).
2
Connect power pins
• 3V3 pin → net label +3V3 (from PSU sheet)
• GND pins (both) → GND power symbol
• 5V pin → net label +5V (can leave unconnected if powering via 3V3 only — add a "No Connect" X flag if unused)
3
Assign GPIO net labels for relay control
Press L to place net labels:

Add "No Connect" flags on unused pins

Also add No Connect on any other D-pins not assigned above.

Add decoupling capacitors on 3V3

SHEET 4 Relay Channel 1 — PC817 + S8050 + SRD-05VDC relay_ch1.kicad_sch
ℹ Signal path: GPIO0 → R (330Ω) → PC817 LED anode → PC817 LED cathode → GND. PC817 phototransistor drives S8050 base → S8050 collector energises relay coil. 1N4007 clamps back-EMF.
RELAY_CH1_IN (GPIO0)
      │
     [R1 330Ω]
      │
      ├── PC817 U4 pin1 (LED Anode)
      │        pin2 (LED Cathode) ── GND
      │
      │   PC817 pin3 (Collector) ─── +5V
      │   PC817 pin4 (Emitter)  ─┬─ [R2 10kΩ] ── GND
      │                          └─ S8050 Q1 Base
      │
      │   S8050 Q1 Collector ────── Relay K1 Coil pin1
      │   S8050 Q1 Emitter  ─────── GND
      │
      │   [D1 1N4007] cathode ──── +5V
      │               anode  ──── Relay K1 Coil pin1
      │
      │   Relay K1 Coil pin2 ───── +5V
      │
      │   Relay K1 COM ──────────── LOAD_L_SWITCHED (to mains output)
      │   Relay K1 NO  ──────────── MAINS_L_RELAY (from mains input)
      │   Relay K1 NC  ──────────── No Connect (X)
1
Place input resistor R1
Press A → Device:R. Ref: R1. Value: 330.
Pin 1 → net label RELAY_CH1_IN
Pin 2 → wire going right to PC817 anode
2
Place PC817 optocoupler U4
Press A → Isolator:PC817 (DIP-4).
• Pin 1 (LED Anode, A) → connect to R1 pin 2
• Pin 2 (LED Cathode, K) → GND
• Pin 3 (Collector, C) → net label +5V (via 10kΩ optional pull-up, see next step)
• Pin 4 (Emitter, E) → wire → S8050 base (see step 4)
⚠ PC817 CTR is ~100%. At 3.3V GPIO, R1=330Ω gives ~9mA LED current (well above 1mA minimum). Output transistor will saturate properly.
3
Add pull-up resistor R2 on PC817 collector
Press A → Device:R. Ref: R2. Value: 10k.
• R2 pin 1 → +5V power symbol
• R2 pin 2 → connects to PC817 pin 3 (Collector) AND S8050 Base junction
4
Place S8050 NPN transistor Q1
Press A → Device:Q_NPN_BCE → search S8050 or use generic NPN (assign S8050 in properties).
• B (Base) → junction: PC817 Emitter + R2 bottom
• C (Collector) → Relay K1 Coil pin (one end)
• E (Emitter) → GND
5
Place flyback diode D1 across relay coil
Press A → Device:D. Part: 1N4007.
Orientation: cathode toward +5V (pointing up), anode toward Q1 collector.
• Anode → Q1 Collector node (same as Relay Coil pin 1)
• Cathode → +5V
6
Place SRD-05VDC-SL-C relay K1
Press A → Relay_THT:Relay_SPDT_Songle_SRD-05VDC-SL-C.
Pin assignment:

7
Add screw terminal for load output
Press A → Connector:Screw_Terminal_01x02. Ref: J2. Value: LOAD_CH1.
• Pin 1 → net label LOAD_CH1 (Relay COM)
• Pin 2 → net label MAINS_N (Neutral)
SHEET 5 Relay Channel 2 — (identical to Ch1) relay_ch2.kicad_sch
ℹ Channel 2 is identical to Channel 1. Simply duplicate the entire Channel 1 sub-circuit and update references and net labels.
1
Duplicate CH1 circuit on a new sheet
Select all of the CH1 schematic. Press Ctrl+C then open relay_ch2.kicad_sch and paste with Ctrl+V.
2
Update all reference designators
Change: R1→R3, R2→R4, U4→U5, Q1→Q2, D1→D2, K1→K2, J2→J3
3
Update all net labels
Double-click each net label and change:
RELAY_CH1_IN → RELAY_CH2_IN
LOAD_CH1 → LOAD_CH2
MAINS_L_SW stays the same (same mains live rail feeds both NO contacts)
4
Verify screw terminal J3
• Pin 1 → LOAD_CH2
• Pin 2 → MAINS_N
SHEET 6 Energy Monitor — HLW8012 / BL0937 energy_monitor.kicad_sch
⚡ DANGER: HLW8012 VP and IP pins are connected directly to mains voltage through a voltage divider. Maintain 5mm minimum clearance from all LV signals on the PCB.
MAINS_L ──[R5 1MΩ]──[R6 1MΩ]──[R7 1MΩ]──[R8 1MΩ]──┬── VP pin (HLW8012)
                                                      └──[R9 1kΩ]── GND

MAINS_L ──[CT / shunt]── IP+ pin (HLW8012)
                    GND ── IP- pin (HLW8012)

HLW8012 VCC ── +5V
HLW8012 GND ── GND
HLW8012 CF  ── HLW_CF  (GPIO20 / D9)
HLW8012 CF1 ── HLW_CF1 (GPIO21 / D3)
HLW8012 SEL ── HLW_SEL (GPIO22 / D4)
1
Place HLW8012 IC symbol
Press A → search HLW8012. If not in library, create a custom symbol with pins:
VCC, GND, CF, CF1, SEL, VP, VN, IP+, IP−
2
Wire power and digital pins
• VCC → +5V power symbol
• GND → GND power symbol
• CF → net label HLW_CF
• CF1 → net label HLW_CF1
• SEL → net label HLW_SEL
Add bypass cap C6 (100nF) from VCC to GND.
3
Build voltage divider on VP pin (HIGH VOLTAGE ZONE)
Place 4× Device:R in series: R5, R6, R7, R8 each = 1MΩ.
Top of R5 → net label MAINS_L
Junction of R8 bottom and R9 top → VP pin of HLW8012
R9 = 1kΩ, bottom → GND
⚠ On PCB: these resistors must be in the HV zone. The HLW8012 chip straddles the isolation boundary — VP/IP pins go toward HV, digital pins (CF/CF1/SEL) go toward LV.
4
Wire current sensing (shunt resistor method)
Place R10 = 1mΩ (or use a proper current transformer).
• R10 pin 1 → net label MAINS_L or inline with load
• R10 pin 2 → IP+ pin of HLW8012
• IP− pin → GND

Alternative (safer): Use a 5A/5mA current transformer (CT) between MAINS_L and load, connecting secondary to IP+/IP−.
SHEET 7 Mains Input, Fuse & MOV mains_input.kicad_sch
⚡ CRITICAL SAFETY: The fuse F1 goes in the LIVE wire only. Never fuse the neutral. The MOV goes between Live and Neutral to clamp surges.
  J_MAINS connector (IEC 3-pin or screw terminal)
  │
  ├── PE (Earth/Ground) ──────────────────────── Chassis GND (PE symbol)
  │
  ├── N (Neutral) ──────────────────────────────── MAINS_N net label
  │
  └── L (Live) ──[F1 10A 250VAC]──┬─────────────── MAINS_L net label
                                   │
                               [MOV1 275VAC]
                                   │
                               MAINS_N (bottom of MOV)
1
Place mains input connector J_MAINS
Press A → Connector:Screw_Terminal_01x03. Ref: J4. Value: AC_MAINS_IN.
Label pins: Pin1=L, Pin2=N, Pin3=PE (Earth)
2
Place fuse F1 in LIVE wire
Press A → Device:Fuse. Ref: F1. Value: 10A 250VAC 5x20mm.
• F1 pin 1 → J4 pin 1 (L input)
• F1 pin 2 → net label MAINS_L
3
Wire Neutral net
J4 pin 2 (N) → net label MAINS_N directly (no fuse on neutral).
4
Add MOV surge protection (optional but recommended)
Press A → Device:Varistor. Ref: RV1. Value: 275VAC MOV.
• MOV pin 1 → MAINS_L
• MOV pin 2 → MAINS_N
5
Wire Earth (PE)
J4 pin 3 (PE) → PWR:Earth power symbol (press P, search Earth). This represents chassis ground, separate from signal GND.
6
Add relay mains input connector J5
For distributing switched mains to loads, add Screw_Terminal_01x02. Ref: J5.
Pin 1 → MAINS_L_SW (connected to relay NO contact)
Pin 2 → MAINS_N

⚠ Critical Safety Rules for Schematic & PCB Do not proceed without reviewing
  • Fuse F1 is placed in the LIVE (L) wire only — never on neutral
  • PC817 provides 5kV optical isolation — this gap must also be reflected as a keepout zone on PCB
  • HLW8012 VP/IP pins carry mains voltage — 5mm minimum clearance from all LV copper on PCB
  • General HV/LV boundary: 3mm minimum clearance between zones
  • All mains-carrying traces on PCB: minimum 2mm trace width (preferably 2.5mm for 10A)
  • AMS1117 SOT-223 thermal tab (Pin 2) must be soldered to a GND copper pour
  • GPIO12, GPIO13, GPIO14 on XIAO must have No Connect flags — these are internally wired to the flash memory chip
  • 1N4007 flyback diodes must be present on both relay coils (K1 and K2)
  • MOV275 clamps L-to-N surges — polarity does not matter but ensure voltage rating ≥ 275VAC
  • Earth/PE pin of J4 must connect to chassis (PCB mounting screw) — do not connect PE to signal GND
