RelaySwitch_C6 — R18 Shunt Resistor Wiring

WSK2512R0100FEA (1mΩ, 4-Terminal Kelvin Sense) + HLW8012

❌ Current Wiring (Incorrect)
✅ Correct Wiring (How It Should Be)
Current path (I1→I2) — high current
Sense path (E1→V1P, E2→V1N) — no current
Relay connections
📋 R18 Pin Connection Summary
Pin Function Current (Wrong) Correct Connection
I1 Current IN (high current pad) Not Connected ✗ → MAINS_L (net "L")
I2 Current OUT (high current pad) Not Connected ✗ → K1 COM + K2 COM (relay loads)
E1 Voltage Sense + (Kelvin) Connected to L ✗ (carrying current!) → HLW8012 V1P (pin 2)
E2 Voltage Sense − (Kelvin) Connected to relay loads ✗ → HLW8012 V1N (pin 3)
Key Principle: In a 4-terminal Kelvin shunt, I1/I2 carry the full load current through thick pads, while E1/E2 are thin sense-only connections that go directly to the HLW8012's differential current input (V1P/V1N). This eliminates PCB trace resistance from the measurement.

Alternative (simpler): If you don't need Kelvin accuracy, remove R18 entirely and use your existing R10 (standard 2-pin 1mΩ 2512) which is already correctly wired to V1P/V1N in your schematic.
