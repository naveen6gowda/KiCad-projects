⚡ HV Zone Components

  • J4 — Mains input connector (L/N/PE)
  • F1 — Fuse T2A 250VAC (Live wire only)
  • RV1 — MOV varistor 275VAC (L to N)
  • U1 input side — HLK-PM01 AC input pins
  • K1/K2 contacts — Relay COM/NO/NC pins
  • U5 V1P/V1N — HLW8012 voltage sense pins
  • R10 — Shunt resistor (in series with Live)
  • J5 — Switched load output connector

✓ LV Zone Components

  • U6 — AMS1117-3.3 LDO regulator
  • U2 — XIAO ESP32-C6 MCU
  • Q1/Q2 — S8050 NPN transistors (relay drivers)
  • D1/D2 — 1N4007 flyback diodes (coil side)
  • R1–R9 — Signal resistors, pull-ups, LED limits
  • C1–C6 — Decoupling capacitors (5V and 3.3V)
  • K1/K2 coil — Relay coil pins (5V side only)
  • J1/J2/J3 — UART debug / programming headers
  • U5 CF/CF1/SEL/GND/VDD — HLW8012 logic pins

⚠ Isolation Boundary Components

  • U4/U5 PC817 — Optocoupler (5kV isolation)
  • LED side → LV zone (3.3V logic from ESP32)
  • Phototransistor side → drives relay coil (5V LV)
  • Min 3mm clearance between LED and transistor pads on PCB
  • U1 HLK-PM01 — Internal isolation ~2500VAC
  • AC input pads: HV zone | DC output pads: LV zone
  • No copper pour crossing the HLK body midline

⚡ PCB Layout Rules

  • HV ↔ LV clearance: min 3mm (IEC60950)
  • HLW8012 V1P/V1N pads: min 5mm from LV copper
  • Mains traces: min 2mm wide (2.5mm for 10A)
  • Relay contact pads: per HF3FF/SRD datasheet creepage
  • No LV copper pour in HV zone
  • Place PE/Earth via to mounting hole
  • HV zone: no silkscreen text (prevents arc tracking)
  • Add keepout zone around PC817 optical gap
